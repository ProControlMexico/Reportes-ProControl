¿Terminar este servicio?

Esta acción guardará todas las capturas registradas en el historial de tendencias y limpiará el panel para iniciar un nuevo reporte de servicio.

¿Cargar servicio para editar?

Esto recuperará las capturas de este servicio al panel principal para modificarlas. Si tienes un servicio activo actual, sus capturas no guardadas se perderán.

¿Desea modificar la configuración?

Se reactivará el formulario de estaciones para que puedas agregar, eliminar o renombrar equipos.

Si hay un servicio activo en curso, las capturas registradas podrían verse afectadas.
